# HTYF Event Booking Platform

A clean-architecture ASP.NET Core MVC application for managing events and bookings, with secure authentication and Memberbase CRM integration.

## Features
- Event listing and details
- Date-based event filtering
- User authentication (ASP.NET Identity)
- Secure event booking for logged-in users
- Memberbase CRM contact synchronization
- Clean Architecture (Domain, Application, Infrastructure, Web)

## Tech Stack
- ASP.NET Core MVC (.NET 7)
- Entity Framework Core
- SQL Server
- ASP.NET Identity
- Memberbase API
- Bootstrap 5

## How to Run
1. Clone the repository
2. Update connection string in `appsettings.json`
3. Add Memberbase API key via User Secrets
4. Run database migrations
5. Start the application
